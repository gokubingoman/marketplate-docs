# Playbook Expansion — Onboarding, Standards & Internal Documentation

Governing prompt for the `docs/` workstream expansion. See [docs/README.md](../docs/README.md) for the live index.

## Workstreams

| Folder | Deliverables |
|--------|--------------|
| `docs/onboarding/` | chef-welcome-package, chef-onboarding-checklist, chef-success-guide |
| `docs/internal/` | company-overview, executive-summary, partner-handbook |
| `docs/standards/` | marketplate-standards, chef-quality-standards, trust-and-safety-standards |
| `docs/help-center/` | Architecture + 22 category articles |
| `docs/customer-success/` | Lifecycle, retention, education, metrics |
| `docs/training/` | 9 role onboarding guides |
| `docs/playbooks/` | 8 cross-functional operational playbooks |

## Agent execution

Run parallel specialist agents per workstream. Each agent reads `company/constitution.md` and cross-links existing docs. No placeholders.

## Writing standard

Production-ready prose as if written by Principal PM, Principal Engineer, Head of Ops, Head of Trust & Safety, Technical Writer, and UX Lead working together.
